# Privacy Policy for TabOrg

TabOrg is a Chrome new tab extension that helps users organize open browser tabs.

TabOrg uses the Chrome tabs permission to read the titles, URLs, favicons, window IDs, and tab IDs of currently open tabs. This information is used only to display and organize tabs on the new tab page, group tabs by domain, switch to selected tabs, and close tabs when the user chooses to do so.

TabOrg uses Chrome storage to save the user's saved list locally in the browser. The saved list may include page titles, URLs, favicons, and saved timestamps.

TabOrg does not collect, sell, share, or transmit user data to any external server or third party. All data is processed locally in the user's browser.

TabOrg does not use user data for advertising, analytics, creditworthiness, lending, or any purpose unrelated to tab organization.

If you have questions, contact the repository owner through GitHub:
